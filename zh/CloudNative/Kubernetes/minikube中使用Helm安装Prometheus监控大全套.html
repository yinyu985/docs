                # Helm

                Helm жҳҜдёҖдёӘз”ЁдәҺ Kubernetes еә”з”ЁзЁӢеәҸйғЁзҪІе’Ңз®ЎзҗҶзҡ„еҢ…з®ЎзҗҶеҷЁгҖӮе®ғзҡ„дё»иҰҒдҪңз”ЁжҳҜз®ЎзҗҶ Kubernetes дёӯзҡ„ ChartsпјҢе®ғд»¬жҳҜдёҖз»„йў„е®ҡд№үзҡ„ Kubernetes иө„жәҗжЁЎжқҝпјҢеҸҜд»ҘеңЁ Kubernetes зҫӨйӣҶдёӯйғЁзҪІгҖӮHelm еҸҜд»Ҙи®©жӮЁиҪ»жқҫең°еҲӣе»әгҖҒе…ұдә«е’Ңе®үиЈ…еҹәдәҺ Kubernetes зҡ„еә”з”ЁзЁӢеәҸпјҢд»ҘеҸҠз®ЎзҗҶе®ғд»¬зҡ„зүҲжң¬е’Ңдҫқиө–е…ізі»гҖӮ

                д»ҘдёӢжҳҜ Helm зҡ„дёҖдәӣж ёеҝғжҰӮеҝөпјҡ

                1. ChartпјҡдёҖдёӘ Chart еҢ…еҗ«дәҶдёҖз»„йў„е®ҡд№үзҡ„ Kubernetes иө„жәҗжЁЎжқҝпјҢе®ғд»¬еҸҜд»ҘдёҖиө·йғЁзҪІдёҖдёӘеә”з”ЁзЁӢеәҸгҖӮдёҖдёӘ Chart еҸҜд»ҘеҢ…еҗ«еӨҡдёӘ YAML ж–Үд»¶пјҢз”ЁдәҺе®ҡд№үйғЁзҪІгҖҒжңҚеҠЎгҖҒй…ҚзҪ®зӯүиө„жәҗгҖӮChart йҖҡеёёеҢ…еҗ«дёҖдёӘ Chart.yaml ж–Үд»¶е’ҢдёҖдёӘ values.yaml ж–Үд»¶пјҢеҲҶеҲ«е®ҡд№үдәҶ Chart зҡ„е…ғж•°жҚ®е’Ңй»ҳи®Өй…ҚзҪ®гҖӮ
                2. RepositoryпјҡHelm Chart еӯҳеӮЁеә“жҳҜдёҖдёӘеҸҜе…¬ејҖи®ҝй—®зҡ„дҪҚзҪ®пјҢе…¶дёӯеҢ…еҗ«дёҖз»„еҸҜз”Ёзҡ„ ChartsгҖӮеӯҳеӮЁеә“йҖҡеёёдҪҝз”Ё HTTP жңҚеҠЎеҷЁжүҳз®ЎпјҢе…Ғи®ёз”ЁжҲ·д»ҺиҝңзЁӢи®ҝй—®е’ҢдёӢиҪҪ ChartsгҖӮ
                3. ReleaseпјҡеңЁ Helm дёӯпјҢRelease жҳҜжҢҮдёҖдёӘ Chart зҡ„е®һдҫӢгҖӮеңЁйғЁзҪІиҝҮзЁӢдёӯпјҢHelm е°Ҷ Chart жёІжҹ“дёә Kubernetes иө„жәҗпјҢе№¶е°Ҷе…¶е®үиЈ…еҲ°зҫӨйӣҶдёӯгҖӮжҜҸдёӘ Release йғҪжңүдёҖдёӘе”ҜдёҖзҡ„еҗҚз§°пјҢе…Ғи®ёжӮЁеңЁйғЁзҪІеӨҡдёӘзүҲжң¬зҡ„зӣёеҗҢ Chart ж—¶еҜ№е®ғд»¬иҝӣиЎҢеҢәеҲҶгҖӮ
                4. ValuesпјҡValues жҳҜ Chart дёӯзҡ„дёҖз»„й»ҳи®Өй…ҚзҪ®пјҢз”ЁдәҺжҺ§еҲ¶еә”з”ЁзЁӢеәҸзҡ„йғЁзҪІе’ҢиЎҢдёәгҖӮе®ғд»¬еҸҜд»ҘеңЁе®үиЈ… Chart ж—¶йҖҡиҝҮе‘Ҫд»ӨиЎҢж Үеҝ—жҲ– YAML ж–Үд»¶иҝӣиЎҢиҰҶзӣ–пјҢд»Ҙе®ҡеҲ¶еҢ– Chart зҡ„йғЁзҪІгҖӮ
                5. TemplateпјҡжЁЎжқҝжҳҜ Chart дёӯзҡ„ Kubernetes иө„жәҗжЁЎжқҝпјҢе®ғд»¬еҸҜд»ҘйҖҡиҝҮ Go дёӯзҡ„ж–Үжң¬жЁЎжқҝиҜӯиЁҖиҝӣиЎҢе®ҡд№үгҖӮHelm дҪҝз”ЁжЁЎжқҝжқҘж №жҚ® Values з”ҹжҲҗ Kubernetes иө„жәҗпјҢе№¶е°Ҷе…¶йғЁзҪІеҲ° Kubernetes зҫӨйӣҶдёӯгҖӮ

                дҪҝз”Ё HelmпјҢжӮЁеҸҜд»ҘиҪ»жқҫең°еҲӣе»әгҖҒжү“еҢ…е’Ңе…ұдә« ChartпјҢд»ҘеҸҠз®ЎзҗҶе®ғд»¬зҡ„зүҲжң¬е’Ңдҫқиө–е…ізі»гҖӮжӮЁиҝҳеҸҜд»ҘдҪҝз”Ё Helm еңЁ Kubernetes зҫӨйӣҶдёӯе®үиЈ…гҖҒеҚҮзә§е’Ңеӣһж»ҡеә”з”ЁзЁӢеәҸпјҢд»ҘеҸҠеңЁеӨҡдёӘзҫӨйӣҶд№Ӣй—ҙе…ұдә« ChartsгҖӮ

                # Prometheus

                Prometheusзӣ‘жҺ§зі»з»ҹжҳҜдёҖз§ҚејҖжәҗзҡ„зӣ‘жҺ§и§ЈеҶіж–№жЎҲпјҢеҸҜз”ЁдәҺзӣ‘жҺ§дә‘и®Ўз®—гҖҒе®№еҷЁе’Ңеҫ®жңҚеҠЎзҺҜеўғдёӯзҡ„еә”з”ЁзЁӢеәҸгҖӮе®ғжңҖеҲқз”ұSoundCloudејҖеҸ‘пјҢзҺ°еңЁз”ұCNCFпјҲCloud Native Computing Foundationпјүз®ЎзҗҶгҖӮ

                PrometheusдҪҝз”ЁPullжЁЎеһӢжқҘж”¶йӣҶжҢҮж ҮпјҢиҝҷж„Ҹе‘ізқҖе®ғд»ҺиҰҒзӣ‘жҺ§зҡ„еә”з”ЁзЁӢеәҸдёӯжӢүеҸ–жҢҮж Үж•°жҚ®гҖӮе®ғж”ҜжҢҒи®ёеӨҡдёҚеҗҢзұ»еһӢзҡ„жҢҮж ҮпјҢеҢ…жӢ¬и®Ўж•°еҷЁгҖҒжөӢйҮҸеҖје’Ңж‘ҳиҰҒгҖӮPrometheusдҪҝз”ЁPromQLпјҲPrometheus Query LanguageпјүжҹҘиҜўиҜӯиЁҖжқҘжҹҘиҜўе’ҢиҒҡеҗҲжҢҮж Үж•°жҚ®пјҢд»ҺиҖҢдҪҝз”ЁжҲ·иғҪеӨҹеҜ№зӣ‘жҺ§ж•°жҚ®иҝӣиЎҢжӣҙж·ұе…Ҙзҡ„еҲҶжһҗе’ҢеҸҜи§ҶеҢ–гҖӮ

                Prometheusе…·жңүи®ёеӨҡй«ҳзә§еҠҹиғҪпјҢдҫӢеҰӮиҮӘеҠЁеҸ‘зҺ°жңҚеҠЎгҖҒеҠЁжҖҒй…ҚзҪ®гҖҒжҠҘиӯҰе’ҢеҸҜи§ҶеҢ–гҖӮе®ғиҝҳе…·жңүе№ҝжіӣзҡ„йӣҶжҲҗиғҪеҠӣпјҢеҸҜдёҺе…¶д»–е·Ҙе…·е’Ңзі»з»ҹйӣҶжҲҗпјҢеҰӮGrafanaгҖҒAlertmanagerе’ҢKubernetesгҖӮ

                жҖ»зҡ„жқҘиҜҙпјҢPrometheusжҳҜдёҖз§ҚеҠҹиғҪејәеӨ§гҖҒзҒөжҙ»е’Ңжҳ“дәҺдҪҝз”Ёзҡ„зӣ‘жҺ§и§ЈеҶіж–№жЎҲпјҢеҸҜеё®еҠ©ејҖеҸ‘дәәе‘ҳе’Ңиҝҗз»ҙдәәе‘ҳжӣҙеҘҪең°зҗҶи§Је’Ңз®ЎзҗҶд»–д»¬зҡ„еә”з”ЁзЁӢеәҸе’ҢеҹәзЎҖи®ҫж–ҪгҖӮ

                еҜ№дәҺдёҖдёӘе·Із»ҸжҲҗеҠҹе®үиЈ…й…ҚзҪ®зҡ„minikubeпјҢжҲ‘д»¬иҝҳйңҖиҰҒе®үиЈ…HelmпјҢеңЁMacOSдёӯпјҢдҪҝз”ЁbrewдҪңдёәеҢ…з®ЎзҗҶеҷЁе®үиЈ…HelmпјҢе°Ҷдјҡйқһеёёж–№дҫҝпјҢеҸӘйңҖдёҖжқЎе‘Ҫд»Ө

                admin/prom-operator,еҪ“з„¶еҰӮжһңдҪ еҫҲжңүиҖҗеҝғзҡ„еңЁminikube dashboardзҡ„з•ҢйқўиҖҗеҝғең°зӮ№дёҠеҮ еҲҶй’ҹпјҢеҲҶеҲ«зңӢзңӢйҮҢйқўйғҪжңүе•ҘпјҢдҪ дјҡеңЁsecretзҡ„еҲҶж ҸйҮҢзңӢеҲ°

                йӮЈд№ҲзҹҘйҒ“дәҶgrafanaзҡ„дҝЎжҒҜпјҢеҰӮдҪ•и®ҝй—®е‘ўпјҹе·ІзҹҘеҲҡжүҚеҲӣе»әзҡ„serviceйғҪжҳҜClusterIPзұ»еһӢзҡ„

                еңЁ Minikube дёӯпјҢеҸҜд»ҘдҪҝз”Ё minikube service е‘Ҫд»ӨжқҘжҡҙйңІ Kubernetes ServiceпјҢе№¶е°Ҷе…¶жҳ е°„еҲ°дё»жңәдёҠзҡ„йҡҸжңәз«ҜеҸЈгҖӮжӯӨж—¶пјҢжӮЁеҸҜд»ҘдҪҝз”Ёдё»жңә IP ең°еқҖе’ҢиҜҘз«ҜеҸЈеҸ·и®ҝй—®иҜҘжңҚеҠЎгҖӮ

                дҫӢеҰӮпјҢеҰӮжһңжӮЁжңүдёҖдёӘеҗҚдёә my-service зҡ„ ServiceпјҢжӮЁеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өе°Ҷе…¶жҡҙйңІеҮәжқҘпјҡ
